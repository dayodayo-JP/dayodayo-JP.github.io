# Scratch架空企業株式取引所計画
　Scratch上に架空企業の株式取引所を建設し、産業活動を活発化させます。概要としては、架空企業が株を売り出しそれを投資者が買う形である為、詳細は株式取引所とは少し異なる。
# 次元・独自通貨への対応
　別次元や同次元の独自通貨に関しては為替ルートを用意しますが、為替ルートは別次元の通常は固定とします。次元には次元の独自通貨がありそれを円に為替すれば良いため、私たちは次元に関しては一切関与致しません。基本的には円による取引であります。為替ルートが変動しない理由は、円取引による過度な円高を防ぐ為です。
　為替ルートは別次元の消費の多さ等と新為替ルート追加の書類に記載される推定為替ルートを元に決定する。
# 資金・株式管理の仕組み
　取引参加の際、個人は社員として働いている企業と働いてきた期間(大体で良い)を、企業は現在の持株数と株を持つ企業、全社員のScratchアカウント名と基本給、企業の現在の架空日本銀行券及びに架空日本円硬貨保有金額(これに関しては取引毎に必要)を報告する義務を負います。個人の取引参加の際に、企業より報告されたScratchアカウント名を照合し、認証を行います。個人は企業から報告される基本給の情報を元に個人の資産を割り出し、それを株式取引に利用するものとする。企業の金額報告には精密さが求められる為、経理部門の業務形式が整っているかも上場条件の一つとなってくるだろう。かつ、真偽不明の粉飾報告は監視の元無くなる為、それは理解して頂きたい。(取引監視システムは存在する)
